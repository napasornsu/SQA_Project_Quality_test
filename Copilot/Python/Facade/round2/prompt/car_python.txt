กำหนดให้ใช้ Facade Design Pattern ในการเขียนโค้ด ตาม Requirement ที่กําหนดให้ โดยเขียนโค้ดด้วยภาษา python
Requirement รถ : Object ประเภทรถยนต์ โดยมีคุณสมบัติเช่น 
เครื่องยนต์: ฟังก์ชันการสตาร์ทเครื่องยนต์ ฟังก์ชันการดับเครื่องยนต์ 
ไฟรถ: ฟังก์ชันการเปิดไฟหน้า ฟังก์ชันการปิดไฟหน้า 
พวงมาลัย: ฟังก์ชันการเลี้ยวซ้าย ฟังก์ชันการเลี้ยวขวา 
เบรก: ฟังก์ชันการเบรก ฟังก์ชันการปลดเบรก
####
จากโค้ดที่สร้างเขียน pytest เพื่อทดสอบโค้ดที่กําหนดให้ โดย Statement coverage เป็น 100% และ Branch coverage เป็น 100%
