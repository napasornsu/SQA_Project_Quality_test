----Factory Method-Python----
กําหนดให้ใช้ Factory Method Design Pattern ในการเขียนโค้ดตามโจทย์ที่กําหนดให้โดยเขียนโค้ดด้วยภาษา Python

ต้องการสร้างระบบร้านกาแฟโดยใช้ Factory Method ในการสร้างเครื่องดื่มกาแฟหลากหลายประเภท ได้แก่ Americano, Latte, Cappuccino, Espresso, และ Mocha 
แต่ละเครื่องดื่มมีวิธีการเตรียมและเสิร์ฟที่แตกต่างกัน แต่โครงสร้างหลักของการสั่งเครื่องดื่มจะยังคงเหมือนกัน ผู้ใช้สามารถเลือกประเภทของกาแฟที่ต้องการสั่งได้ 
โดยระบบจะสร้างกาแฟตามที่เลือกและเตรียมพร้อมเสิร์ฟให้

จากโค้ดที่สร้างเขียน pytest เพื่อทดสอบโค้ดที่กําหนดให้ โดย Statement coverage เป็น 100% และ Branch coverage เป็น 100%
